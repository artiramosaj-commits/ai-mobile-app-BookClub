A mobile application for tracking your reading list, rating books, and writing reviews. Built with React Native (Expo) and Supabase.

## Features
- **My Library**: See all your books in one place.
- **Status Tracking**: Mark books as 'Want to Read', 'Reading', or 'Completed'.
- **Interactive Ratings**: Rate books 1-5 stars.
- **Reviews**: Write thoughts on your reads.
- **Filter**: Easily sort through your library.

## Getting Started

1. `npm install`
2. Configure Supabase credentials in `src/lib/supabase.ts`.
3. `npx expo start`
